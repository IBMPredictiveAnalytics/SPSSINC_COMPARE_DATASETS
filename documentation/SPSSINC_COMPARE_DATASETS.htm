Compare Datasets

This procedure compares the cases and/or variable dictionaries of two open datasets. Both must have names.

Check the Compare cases box to compare the data in the two datasets. Check the Compare dictionary properties box to compare selected properties of the variables. If neither box is checked, there will be no output.

Enter the dataset names of the primary and secondary datasets in the boxes provided. Any requested case comparison information will be written to the primary dataset.

If comparing cases, enter the name of a case id variable present in both datasets. Both datasets must be sorted ascending by the ID variable.

Enter the name of a new variable in the New Variable to Hold Difference Count field to produce a variable whose values are the number of differences in the variables compared for each case. If a case from the primary dataset is not present in the secondary dataset, the count value will be missing.

A summary report of the differences is written to the Viewer, but the details are only written to the primary dataset.

If a root name is entered in the Root Name for New Variables field, a new variable with a name starting with the specified root will be created for each variable compared. The values will be 0 or 1 according to whether the variable values are equal or not.

Enter a filename in the Log file file to produce a text file listing the detailed differences between the two files. It will list case ids present in only one dataset and all value differences for the selected variables.

If the variable dictionaries are to be compared, check one or more properties to compare. A report will be written to the Viewer showing where any differences are.

Additional Features

This dialog generates syntax for the SPSSINC COMPARE DATASETS extension command. To display help for this command, run the following syntax:

SPSSINC COMPARE DATASETS /HELP.

Requirements

This dialog requires the Integration Plug-In for Python and the SPSSINC COMPARE DATASETS extension command. For IBM® SPSS® Statistics 19 and higher, the Plug-In and the extension command are installed with the Essentials for Python package. For more information, see How to Get Integration Plug-Ins, under Core System > Frequently Asked Questions in the IBM SPSS Statistics Help system. If you downloaded SPSSINC COMPARE DATASETS from the SPSS Community, then please follow the instructions in the associated readme file.
